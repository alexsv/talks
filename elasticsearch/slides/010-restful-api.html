restful api

  
how to create a document

  # `cars` - index name
  # `sedan` - document type
  # 1 - document id
  
  curl -XPOST \`http://localhost:9200/cars/sedan/1` -d
  {  
    manufacturer: "Audi",
    model: "A4"
  }

  
how to update
    
  curl -XPUT \`http://localhost:9200/cars/sedan/1` -d
  {  
    manufacturer: "Audi",
    model: "A6",
    seats: 4
  }
